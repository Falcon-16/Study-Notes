What are Functions & Relations?

A relation is any set of points on a Cartesian plane. Anything on a Cartesian plane is a relation, including inequalities.

A function is a relation with at most one -coordinate for each -coordinate.

Types of Relations

One-to-One Relation

One x value to one y value. It is a function.

One-to-Many Relation

One x value to several y values. It is not a function

Many-to-One Relation

Several x values to one y value. It is a function

Many-to-Many

Several x values to several y values. It is not a function.

Piece wise Function

A piecewise function is just a function with different parts. It is a function with a few functions stuck together.

It is continuous if there is a continuous flow, there is no stop. In simple terms, if you can draw it without picking up your pen, it is continuous. The last function has to end where the next takes off.

It is discontinuous if one function does not join where the other left off, then it is discontinuous.
